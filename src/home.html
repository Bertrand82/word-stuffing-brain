Word‑Stuffing‑Brain is an application designed to help you review, memorize and listen to English vocabulary lists in a friendly and shared environment.

The environment can be shared, via Google Drive, between your computers and mobile phones. It can be also optionally shared with your contacts.

So long the application is not verified by google, to use the Google Drive features, the administrator need to allow the application to access your Google Drive account.
admin drive
Note: The application does not store any of your personal data, it only uses the Google Drive API to read and write files in your Google Drive.
